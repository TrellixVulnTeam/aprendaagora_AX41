


Elementos/Informações/Dados para exibir na publicação no modal

	[ ] - Lista de publicações semelhantes 
	[x] - Opção de editar publicação para o autor da publicação
	[x] - Opção de apagar publicação para o autor da publicação
	[ ] - Opção de encerrar discussão para moderador

	[x] - Título da publicação
	[x] - Tags da publicação (incluindo idioma!)
	[x] - Data da publicação
	[x] - Conteúdo da publicação
	[x] - Nome de usuário do autor da publicação
	[x] - Avatar do autor da publicação
	[x] - Botão de dar 'amei' na publicação
	[ ] - Número de 'ameis' na publicação
	
	[x] - Formulário para comentar na publicação
	[x] - Botão para enviar comentário
	[x] - Comentários escritos na publicação
	[x] - Nome de usuário e link para página dos autores dos comentários
	[x] - Avatar dos autores dos comentários
	[x] - Número de 'amei' dos comentários
	[x] - Botão de dar 'amei' no comentário
	[x] - Opção de editar comentário para autor do comentário
	[x] - Opção de apagar comentário para autor do comentário


Funcionalidades para implementar na publicação no modal

	[x] - Exibir aviso quando as alterações em uma publicação forem salvas

	[ ] - Exibir um modal/popup para confirmar exclusão da publicação quando o usuário clicar em apagar a publicação

	[ ] - Links nas tags para a lista de publicações daquela tag em particular

	[ ] - Registrar 'amei' no comentário quando o ícone do coração for clicado

	[ ] - Exibir número de 'ameis' no comentário

	[ ] - Apagar comentário quando o botão de apagar for clicado

	[ ] - Abrir formulário de edição de comentário quando o botão de edição for clicado
